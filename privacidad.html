Política de Privacidad: Cumpleaños de Andy Última actualización: 11 de mayo de 2026 Esta Política de Privacidad describe cómo se gestiona la información en la aplicación móvil Cumpleaños de Andy (en adelante, "la Aplicación"). La Aplicación es una WebApp (PWA) diseñada exclusivamente para centralizar actividades y contenidos relacionados con una celebración personal. 1. Recopilación de Información La Aplicación no recopila, almacena ni comparte ningún tipo de dato personal de los usuarios. No se requiere registro de cuenta. No se solicitan correos electrónicos, nombres o números de teléfono. No se accede a la ubicación del GPS, contactos o archivos del dispositivo. 2. Tecnologías y Cookies Al ser una aplicación basada en tecnologías web, se utilizan los siguientes elementos técnicos para su funcionamiento: Service Workers: Se utiliza el archivo sw.js para permitir que la aplicación funcione a pantalla completa y sea instalable en el dispositivo. Caché Local: La aplicación puede almacenar temporalmente archivos (imágenes, iconos y estilos) en la memoria caché de su navegador para mejorar la velocidad de carga y permitir el acceso básico sin conexión. 3. Servicios de Terceros La Aplicación integra contenidos de plataformas externas para mejorar la experiencia del usuario. Estos servicios pueden tener sus propias políticas de privacidad: Elfsight: Utilizado para el widget de cuenta atrás. Genially: Utilizado para visualizar la invitación y el juego "Escape Room". GitHub: El código fuente y los archivos se alojan en esta plataforma. 4. Uso por Menores La Aplicación es apta para todas las edades y no contiene contenido inapropiado ni compras integradas. Dado que no se recopilan datos personales, es segura para su uso por menores de edad. 5. Seguridad Aunque la Aplicación no maneja datos sensibles, se distribuye bajo el protocolo seguro HTTPS para garantizar que la conexión entre el dispositivo y el servidor esté cifrada. 6.
